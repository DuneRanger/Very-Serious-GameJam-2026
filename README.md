# Catsino

## Main Description

This is serious! You are in financial ruin and need money fast!
And there is no better way to earn a lot of money quickly, than visiting the CATSINO!

Place bets, collect different items, stir the odds in your favour in this fun roguelike roulette game and win big!

This game uses a procedurally generated roulette, that can be modifiable through items you buy in the shop. You can add and erase numbers or change colors and sizes of individual cells. These decicions will help you improve your chances at winning.

The game ends when you solve all of your financial problems (10 rounds, with an extra option of endless mode)

\
This project was made for The very serious Juniper dev game jam, with the theme: Spin to win!
